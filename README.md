# ai-tools

A public toolbox of small AI-native utilities: CLI scripts, MCP servers, reusable skills, and operating rules for building practical agentic workflows.

## What’s here

- **cli/** — small command-line tools
- **mcp/** — MCP servers and integrations
- **skills/** — reusable task playbooks and prompts
- **rules/** — coding, writing, and agent behavior rules
- **templates/** — starter scaffolds for new tools
